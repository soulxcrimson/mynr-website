Command Pages

Everything MyNr can do

Earning & Mining

Grind coins, mine credits and loot with a progression-based mining system.

  • .daily — Free coins + credits
  • .chest — Store / sell items
  • .explore <amount> — Explore caves
  • .dungeon <amount> — Dungeon run
  • .mine — Start mining
  • .pick — Forest items
  • .rest — Regenerate credits
  • .pray — Random buffs
  • .slot — Spin slots (12h)

Tools & Work

Upgrade pickaxes, repair gear and run jobs that pay in MyNr coins.

  • .equip <tool> — Equip tool
  • .repair — Repair tool
  • .shop — Buy tools/items
  • .workshop — Jobs list
  • .applyjob <ID> — Apply job
  • .resignjob <ID> — Resign job
  • .work — 12h shift
  • .hardwork — 24h shift

Player Info

Track stats, leaderboard position and account balance.

  • .stats — View stats
  • .rank — Mining rank
  • .leaderboard — Leaderboard
  • .balance — Check balance

Items & Buffs

Manage your consumables and check active boosters.

  • .buffs — Active buffs
  • .use <item> — Use items

Social & Reputation

Build trust with vouches and reputation leaderboards.

  • .vouch @user <detail> — Add vouch
  • .rep @user — View rep
  • .rep lb — Rep leaderboard
  • .rep rm @user <index> — Remove fake vouch
  • .report @user <reason> — Report user

Payout System

Robux-style payouts powered by MyNr coins.

  • .payout @user <amount/item> — Create payout ticket
  • .gift @user <amount> — Gift coins

Staff & Extras

Moderation, economy in tickets & secrets

Staff Management

Ticket-centric tools built for moderators.

  • .manage — Staff commands list
  • .warn @user <reason> — Warn
  • .unwarn @user — Unwarn
  • .to @user <10s|10m|10h|10d> — Timeout
  • .unmute @user — Remove timeout
  • .kick @user <reason> — Kick
  • .ban @user <reason> — Ban
  • .unban <user_id> — Unban
  • .case @user — View warnings/reports
  • .approve / .deny — Approve / deny ticket
  • .close — Close ticket

Ticket-Safe Economy

Commands allowed inside payout / report tickets.

  • .balance
  • .gift @user <amount>
